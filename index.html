How It Works

  1. Load Categories File
    Click "Load Categories" button and select your categories .ltx file (from the Grounded Trading System addon). This file contains a list of all game items organized by categories. The left panel will populate with items grouped by category.
  2. Load Item Names (Optional but recommended)
    Click "Load Names (.xml)" button and select one or more string table XML files (available in the tool's repository). In-game translated names will now appear alongside item IDs.
  3. Create a New Trader
    Enter a trader name in the input field (e.g., "sidorovich") and click "+ Add" button.
  4. Add Tiers to Trader
    In Stalker Anomaly, each trader has a list of supplies tiers that usually get unlocked by improving the goodwill with that trader's faction. Check the unpacked game files to see how many supplies tiers a specific trader has. Enter a tier name (e.g., "supplies_1") and optionally add a parent tier name for inheritance (e.g., "supplies_1" as parent for "supplies_2"). Click "+ Tier" and repeat for each tier the trader has.
  5. Add Groups to Tier
    Enter a group name and click "+ Group". You can add multiple groups per tier.
  6. Add Items
    Click on any group header to select it. Browse or search items in the left panel and click any item to add it to the selected group. In the middle panel, you can adjust amount and chance values inline.
  7. Export
    Select trader from dropdown in right panel and click "Copy" or "Download .ltx".
  8. Import Existing Traders (Optional)
    To edit an existing trader, enter the trader name first, then click "Import" and select the trader's .ltx file from the configs/items/trade/ folder in your unpacked game files. The tool will load all tiers, groups, and items from that file.
